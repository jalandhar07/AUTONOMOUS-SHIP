# FPGA-Based Autonomous Ship for Real-Time Navigation and Object Detection

## Overview

The FPGA-Based Autonomous Ship is an intelligent maritime navigation system developed using an EDGE FPGA Board, Raspberry Pi 4B.

The system continuously captures real-time video using a camera, detects obstacles using the YOLOv3-tiny object detection algorithm, measures nearby distances with ultrasonic sensors, and determines safe navigation paths using GPS and path-planning techniques.

The FPGA performs high-speed parallel processing for low-latency object detection, while the Raspberry Pi handles system control, cloud communication, and remote monitoring through RealVNC.
